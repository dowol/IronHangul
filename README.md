# IRONHANGUL
  Read in: **한국어** | [English](./README.en.md)

**IronHangul**(*아이언한글*)은 C#, VB.NET, Unity 등 .NET 플랫폼에서 한글 텍스트 처리를 위한 기능을 구현한 
라이브러리입니다. 쉬운 사용법을 바탕으로 .NET 애플리케이션에 자연스러운 한국어 UI/UX를 제공하는 것을 목표로 개발하는
중입니다.   

> [!IMPORTANT]  
> 현재 **IronHangul**은 개발 진행중으로 아직 베타버전 및 정식버전이 배포되지 않았습니다.

## 소개
한국어 인터페이스를 제공하는 프로그램을 개발할 때, 한글 및 한국어 문법의 특성 때문에 여러 문제점을 겪는 경우가 있습니다.
변수(파라메터)가 있는 메시지를 작성할 때, 초성 검색을 할 때 복잡한 알고리즘을 구현하거나 아예 해당 기능을 포기하는 경우도
부지기수입니다.  
**IronHangul**은 한글 텍스트를 다루는 프로그램에서 자주 사용되지만 구현하기 까다로운 알고리즘을 .NET 플랫폼에서도 빠르고
편리하게 사용할 수 있도록 하는 라이브러리입니다. 상당수의 기능이 .NET에서 제공하는 기본 문자(char) 및 문자열(string)
클래스의 확장 메소드로 구현되어 간편한 조합으로 현대 한글을 다루는 모든 케이스에 대응할 수 있습니다.

## 특징
**IronHangul**은 한글 및 한국어 텍스트를 다루는 프로그램에서 발생하는 복잡한 작업이나 문제점을 해결하기 위한 사항을 
고려하여 제작되었습니다.

### 모든 현대 한글 자모 및 음절 조합을 지원
현대 한글 자음 조합 19자와 모음 조합 21자, 현대 한글 음절 11,172자를 모두 지원하며, 모든 텍스트는 유니코드로 처리됩니다.

### OS별 한글 텍스트 표현 방식을 모두 고려
글자를 컴퓨터에서 표현하기 위해 가장 보편적으로 사용하는 유니코드는, 한글 음절 조합이나 로마자의 다이어크리틱(&acirc;
위에 있는 `^` 같은 부호) 같이 논리적으로 조합하여 사용되는 문자의 경우 표현하는 방법이 2가지인 경우가 종종 있습니다. 기본적
으로는 각각의 문자 코드를 부여하여 둘을 함께 쓰지만[^NFD/조합형], 자주 쓰이는 조합의 경우 메모리 절약을 위해 해당 조합을
합쳐서 하나의 문자 코드를 부여합니다[^NFC/완성형]. 서로 일장일단이 있는 체계여서 표준 없이 각 OS별로 NFC나 NFD 중 하나를
택하여 쓰다보니 OS 간 한글 텍스트를 주고받을 때 문제가 생겨버리는 경우가 있습니다. (정렬 순서가 꼬이거나, macOS에서 
Windows로 파일을 전달할 때 한글이 전부 풀어지는 현상`예) 한글 -> ㅎㅏㄴㄱㅡㄹ`이 대표적인 예시)  

**IronHangul**은 기본적으로 모든 텍스트를 **NFC** 방식으로 처리하며, OS별 기본 설정에 따라, 또는 프로그래머의 설정에
따라 최종 출력 텍스트를 NFC 또는 NFD 방식으로 통일하여 위와 같은 문제를 방지합니다.

### 변수(파라메터)에 따른 한국어 문법적 특성 (조사) 반영
외국인이 한국어를 배울 때 가장 어려워 하는 문법, 우스갯소리로 '한글이 쉽다고 했지 한국어가 쉽다고는 하지 않았다' 라는 말이
나올 정도로 골머리를 앓는 **조사**(助詞), 프로그래밍에서도 조사 때문에 '~은(는)', '~이(가)' 처럼 그다지 자연스럽지 않은 
문장을 반 강제로 써야 했던, 약간 골치 아픈 상황이 있습니다.  
핀테크 앱 **토스**의 개발사에서는 해당 문제를 해결하는 자바스크립트 라이브러리
[es-hangul](https://github.com/toss/es-hangul)을 개발하여 오픈소스로 제공하고 있습니다. 이 편리한 기능을 
.NET에서도 사용할 수 있도록 **IronHangul**에도 도입, 구현하였습니다.

### 기타 편리한 유틸리티 제공
위의 기능 외에 한글 텍스트 전산 처리 시 유용한 몇몇 기능(로마자 표기, 한자, 옛한글)을 구현하거나 구현 중입니다.

## 지원하는 .NET 플랫폼
**IronHangul**은 .NET Standard 2.0으로 개발 중이며, 아래의 플랫폼과 호환됩니다.

* **.NET**: 5 이상
* **.NET Core**: 2 이상
* **.NET Framework**: 4.6.1 이상
* **Mono**: 5.4 이상
* **UWP**: 10.0.16299 이상
* **Unity**: 2018.1 이상

## 라이브러리(패키지) 목록
* `IronHangul`  
  기본적인 기능을 구현한 라이브러리입니다.
* `IronHangul.Romanization`  
  한글(한국어) 텍스트의 로마자 변환을 처리합니다.
* `IronHangul.Hanja`  
  한글과 한자 간 상호작용을 처리합니다.
* `IronHangul.OldHangul`  
  옛한글을 처리하는 라이브러리입니다.
